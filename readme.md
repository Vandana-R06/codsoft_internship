# Advanced Scientific Calculator Web App

## 🚀 Overview
A modern, responsive, and fully functional **Scientific Calculator** built using HTML, CSS, and JavaScript with advanced mathematical capabilities.

---

## ✨ Features

### Basic Operations
- Addition, Subtraction, Multiplication, Division
- Percentage (%)
- Decimal support
- Clear (C), All Clear (AC)
- Backspace (⌫)

### Scientific Functions
- Square (x²), Square Root (√)
- Power (xʸ)
- Logarithm (log, ln)
- Exponential (eˣ)
- Factorial (!)
- Absolute value
- Trigonometry:
  - sin, cos, tan
- Degree / Radian toggle

### Advanced Features
- Memory: MC, MR, M+, M-
- History panel
- Keyboard support
- Error handling

---

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6)
